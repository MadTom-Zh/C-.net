


ColorExpertDialog
以多种配色模式来显示和调节颜色，每个面板可以单独展开或收起；
可设定多个工作颜色，以及设定颜色的透明度；


FileFolderSelector
在使用okki提供的选择文件夹窗口前（像选择文件那样选择文件夹），使用这个窗口选择多个文件和/或文件夹；
注意需要提前设定模式；


FontDialog
可视化选择字体字号、加粗、下划线等等；


MessageBoxEx
除了显示文本和图标，还可以设定选择复选、单选等项目，方便在一个询问窗口中让用户做出更多选择；


ScreenSelectorSimple
定格当前窗口画面，让用户选择一块区域，确认后返回这块区域的图片；
只选区区域的图片，不含涂鸦相关功能；


WindowNotify
一个单纯的工具窗口，可设置窗口标题和窗口内的大号文本；
一般用于提示较长时间的工作状态，使用showdialog阻断窗口操作，等待完成后程序执行窗口关闭；


MultiFilesProperties
显示多个文件夹和/或文件的常见属性，包括文件名，后缀，尺寸，创建时间，修改时间，访问时间，是否只读，是否隐藏；
* 不能显示（一个或多个）磁盘的属性；
窗口中，可以 批量 修改 文件名（仅限前缀），批量 修改 三种时间，只读和隐藏属性；
* 当存在修改（需要“应用”）的条目时，显示为蓝色；修改但时间逻辑错误，显示为橙色；默认存在时间错误，显示为黄色；
