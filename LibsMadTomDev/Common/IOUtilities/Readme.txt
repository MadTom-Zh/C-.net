
更新于 2022-02-28
项目用于 .NET 框架的工程



FileFilterHelper
用于快速解析和设置文件过滤，使用字符串匹配过滤文件；


IOInfoShadow
将文件、文件夹信息复制，避免文件夹、文件被原始信息实体占用；


IOUtilities
多种文件操作的包装，特性：
CSharpWapper，CS封装，文件操作，带错误反馈；
CSharpIOHelper，CS封装，文件操作，带有进度汇报，可中途取消；
Shell32SHFileOperation，Shell32封装，文件操作，获取文件类型描述；
MSVBFileOperation，VB封装，文件操作；
Checker，检查文件路径是否合法；
Other，快速获取一些常用路径，以快捷方式设置和取消开机启动，相对、绝对路径转换；
ClipBoard，代复制和移动标记，设置文件拖动，获取文件投放；


Network
网络相关操作，特性：
Connection，，，，，，
Hosts，获取共享的计算机，和共享计算机中的共享文件夹；
